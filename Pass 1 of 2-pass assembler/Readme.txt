Execute Main.java
Write input (.asm code) in input.asm
Format for input file
	label <tab> mnemonic <tab> operand1,operand2(if any)
	No white spaces between operands
Intermediate code is written into IC.txt
Literal table is written into LITTAB.txt
Pool table is written into POOLTAB.txt
Symbol table is written into SYMTAB.txt
